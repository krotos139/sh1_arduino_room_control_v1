Room Z1
=======

Проект Arduino для контроллера комнаты сситемы "Умный дом".

Реализует следующие функции:
* Управление по Modbus RTU поверх USB serial
* Измерение температуры с помощью датчика DHT22
* Измерение влажности с помощью датчика DHT22
* Измерение загозованности помещения (угарный газ, природный газ) с помощью датчика MQ2
* Детектирование движения с помощью инфракрасного сенсора
* Измерение тока
* Управление 8 реле
* Универсальное управление бытовой техникой через ИК-передатчик
* Управление RGB светодиодной лентой

Пользовательский интерфейс и удленное управление системой реализовано в сопрягаемом планшетном компьютере с ПО sh1_android_room_control_v1
